# Ringbar

> Progressbar but circular

## Usage

### Simple Usage

<preview class="space-x-4">
  <p-ringbar value="15" />
  <p-ringbar value="25" />
  <p-ringbar value="50" />
  <p-ringbar value="75" />
  <p-ringbar value="100" />
</preview>

```vue
<template>
  <p-ringbar value="15" />
  <p-ringbar value="25" />
  <p-ringbar value="50" />
  <p-ringbar value="75" />
  <p-ringbar value="100" />
</template>
```

## API

### Props

| Props           |   Type    | Default | Description                                    |
|-----------------|:---------:|:-------:|------------------------------------------------|
| `size`          | `String`  |  `md`   | Ring size, valid value: `xs`, `sm`, `md`, `lg` |
| `min`           | `Number`  |   `0`   | Lowest progress value                          |
| `max`           | `Number`  |  `100`  | Highest progress value                         |
| `value`         | `Number`  |   `-`   | Progress value                                 |
| `indeterminate` | `Boolean` | `false` | Enable indterminate mode                       |
