## API

### Props

| Props           |   Type    | Default | Description                                    |
|-----------------|:---------:|:-------:|------------------------------------------------|
| `min`           | `Number`  |   `0`   | Lowest progress value                          |
| `max`           | `Number`  |  `100`  | Highest progress value                         |
| `value`         | `Number`  |   `-`   | Progress value                                 |
| `indeterminate` | `Boolean` | `false` | Enable indterminate mode                       |
